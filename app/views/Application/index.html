JSwar

What?

JSwar est un jeu de tanks pour développeurs Javascript. Le but est de détruire les autres tanks, en utilisant son canon pour les repérer et les bombarder. Là où cela devient intéressant, c'est que chaque robot est programmé en Javascript...

How?

Chaque tank est isolé dans un WebWorker. Le jeu communique avec les tanks via des messages, et vice-versa. Vous devez donc écouter les messages que vous envoie le jeu. Aidez-vous du code de base, et regardez les méthodes publiques dans ce fichier: robot.js

Vous pouvez coder et tester le jeu dans l'éditeur, puis publier votre robot lorsqu'il marche comme vous le vouliez.

Where?

Cliquez ici pour accèder à l'éditeur, et commencer à jouer. Gardez un oeil sur l'écran de TV pour voir les robots s'affronter !
